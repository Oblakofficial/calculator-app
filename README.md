# Data Analytics Dashboard

A modern, responsive data analytics dashboard built with HTML, CSS, and JavaScript. Features include:

- Interactive charts and graphs using Chart.js
- Real-time data updates
- Dark/Light theme support
- Data filtering and search capabilities
- Export functionality
- Responsive design for all devices

## Features

- **Interactive Charts**: Line charts, bar charts, and doughnut charts for data visualization
- **Key Metrics**: Overview of important metrics with trend indicators
- **Data Table**: Detailed analytics with search and filter capabilities
- **Theme Support**: Toggle between light and dark modes
- **Export**: Download data in CSV format
- **Real-time Updates**: Automatic data refresh every 30 seconds

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6+)
- Chart.js for data visualization
- Inter font for typography

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start exploring the dashboard!
